================================================================================
                        BROKEN PROVINCES: THE EMPTY THRONE
                           PS1-Style Action RPG Demo
================================================================================

Thank you for playing Broken Provinces! This is an early demo build of a
PS1-era styled open world action RPG. Please report any bugs or feedback to:

    Bug Reports: calebdickenson@gmail.com

================================================================================
                                  CONTROLS
================================================================================

MOVEMENT
--------
    W / S / A / D       Move Forward / Backward / Left / Right
    Shift (Hold)        Sprint (drains stamina)
    Space               Jump
    C                   Crouch / Stealth Mode
                  
COMBAT
------
    Left Mouse Button   Attack (melee) / Cast Spell (if spell equipped)
    Right Mouse Button  Heavy Attack
    Q                   Block
    F                   Lock-On to Target

INTERACTION
-----------
    E                   Interact with NPCs, doors, chests, items

HOTBAR & ITEMS
--------------
    1-9, 0              Use items/spells in hotbar slots 1-10
    T                   Open Wait Menu (pass time)

MENUS & UI
----------
    Tab                 Open Game Menu (inventory, stats, quests, map)
    Escape              Pause Menu / Close current menu
    Z                   Previous Tab (in menus)
    X                   Next Tab (in menus)

CAMERA
------
    Mouse               Look Around (first-person view)
   

================================================================================
                              GAME FEATURES
================================================================================

CORE GAMEPLAY
- First-person PS1-era aesthetic with affine texture warping and vertex jitter
- Real-time combat with light/heavy attacks, blocking, and dodging
- Magic system with learnable spells and mana management
- Ranged combat with bows, crossbows, and muskets (requires ammo)
- Stealth mechanics with crouching and visibility system

CHARACTER PROGRESSION
- Level up and allocate stat points (Grit, Agility, Will, Knowledge, etc.)
- Multiple skills to improve (Stealth, Arcana, Engineering, Dodge, etc.)
- Race selection (Human, Elf, Dwarf, Halfling) affects starting stats

OPEN WORLD
- Daggerfall-style cell streaming for seamless world traversal
- Multiple towns and settlements to explore (Elder Moor, Dalhurst, Thornfield)
- Dungeons and caves with enemies and loot
- Day/night cycle affecting gameplay

NPC SYSTEMS
- Topic-based conversation system with NPCs
- Faction reputation that affects how NPCs treat you
- Quest system with main quests, side quests, and bounties
- Merchants for buying/selling equipment

CRAFTING & ECONOMY
- Crafting system for weapons, armor, and consumables
- Equipment durability and repair stations
- Encumbrance system (no inventory limit, but weight penalties)

SAVE SYSTEM
- Multiple save slots
- Autosave (every 30 seconds and on exit)
- Quicksave support
